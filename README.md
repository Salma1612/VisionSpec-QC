# VisionSpec QC – Intelligent Quality Control System

## 📌 Project Overview

VisionSpec QC is a computer vision-based quality control system designed to detect defective products using machine learning techniques.

## 🎯 Objective

To automate industrial inspection and reduce manual quality checking errors using AI-based image classification.

## 🛠️ Technologies Used

* Python
* OpenCV
* TensorFlow / PyTorch
* NumPy
* Google Colab

## ⚙️ Project Workflow

1. Image Data Collection
2. Data Preprocessing
3. Model Training
4. Defect Classification
5. Performance Evaluation

## 📊 Results

The trained model successfully classifies defective and non-defective products with high accuracy.

## 🚀 How to Run

1. Open the notebook in Google Colab
2. Install required libraries
3. Run all cells
